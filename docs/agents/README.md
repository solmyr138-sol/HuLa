## Prerequisites

- **GitHub CLI** (`gh`) for `to-issues`, `to-prd`, and `triage`. Install from https://cli.github.com/
- **Login once** (in a terminal, from repo root):

  ```powershell
  gh auth login
  ```

  Choose GitHub.com → HTTPS → authenticate via browser.

- **Create triage labels** (after login):

  ```powershell
  .\docs\agents\create-triage-labels.ps1
  ```

  Or manually:

  ```powershell
  gh label create needs-triage --color FEF2C0 --description "Maintainer needs to evaluate"
  gh label create needs-info --color D4C5F9 --description "Waiting on reporter"
  gh label create ready-for-agent --color C2E0FF --description "Ready for AFK agent"
  gh label create ready-for-human --color BFDADC --description "Requires human implementation"
  gh label create wontfix --color E4E4E4 --description "Will not be actioned"
  ```

If `gh` is not recognized, use the full path (default install):

```powershell
& "C:\Program Files\GitHub CLI\gh.exe" auth login
.\docs\agents\create-triage-labels.ps1
```
